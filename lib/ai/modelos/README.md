# Modelos oficiais dos 3 DOCX do Revisor de Defesas

Estes ficheiros são a estrutura obrigatória para a geração dos três documentos:

- **MODELO_PARECER_EXECUTIVO.txt** — Doc 1: Parecer Executivo / Avaliação da Defesa
- **MODELO_ROTEIRO_ADVOGADO.txt** — Doc 2: Roteiro de Audiência — Advogado
- **MODELO_ROTEIRO_PREPOSTO.txt** — Doc 3: Roteiro de Audiência — Preposto

Ao criar um documento cujo título contenha o nome do tipo (ex.: `ROTEIRO_ADVOGADO_-_Caso_X`, `PARECER_EXECUTIVO`, `AVALIACAO_DEFESA`), o sistema usa o modelo correspondente e instrui o LLM a preencher mantendo a estrutura.

Origem: modelos Word (.docx) fornecidos pelo utilizador; o texto foi extraído e guardado aqui para uso em runtime.
